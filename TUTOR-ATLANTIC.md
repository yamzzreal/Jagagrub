Cara Upgrade H2H di Aplikasi Atlantic Pedia
Ikuti langkah-langkah berikut untuk mengajukan upgrade akun ke H2H:

1. Download Aplikasi Atlantic Pedia
Unduh dan instal aplikasi Atlantic Pedia melalui Playstore.

2. Registrasi Akun
Buka aplikasi, lalu buat akun baru jika belum memiliki akun.

3. Login ke Akun Anda
Masuk menggunakan email dan password yang telah didaftarkan.

4. Masuk ke Menu Profile
Setelah login, klik ikon Profile di bagian bawah aplikasi.

5. Buka Menu "Account" â†’ "Pengaturan Profil"
Pilih menu Account, lalu klik Pengaturan Profil.

6. Klik "Selengkapnya" di Bagian Upgrade H2H
Scroll ke bawah dan cari bagian Upgrade H2H, lalu klik Selengkapnya.

7. Isi Formulir Upgrade
Lengkapi formulir sesuai data Anda.

Contoh Pengisian Formulir:
https://files.catbox.moe/ch5sty.jpg

Catatan Penting:
Wajib mencantumkan nomor bot WhatsApp yang aktif, karena admin Atlantic akan mengecek bot Anda terlebih dahulu sebelum menyetujui permintaan upgrade.

Cara Mengambil API Key & Mengatur API di H2H Atlantic Pedia
Ikuti panduan berikut untuk mendapatkan API Key dan melakukan pengaturan API:

1. Login ke Website H2H Atlantic Pedia
Buka link berikut dan login dengan akun Anda:
https://atlantich2h.com

2. Buka Menu API
Setelah login, klik ikon tiga titik di pojok kanan atas.
Pilih menu API dari daftar yang muncul.

3. Salin API Key
Di halaman API, salin API Key yang tertera untuk digunakan dalam integrasi bot atau aplikasi Anda.

4. Atur IP Akses
Pada kolom IP Address, isi dengan:
0.0.0.0

5. Request OTP
Klik tombol Request OTP.
OTP akan dikirim ke email terdaftar Anda.

6. Cek Email & Verifikasi
Buka email Anda dan salin kode OTP yang dikirimkan oleh Atlantic.

7. Simpan Perubahan
Masukkan OTP di kolom yang tersedia, lalu klik Simpan Perubahan.
